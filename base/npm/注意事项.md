## package.json 文件的注意事项
- name 属性（包名）的注意点：
1、包名不能包含中文字符
2、包名不能包含大写

- version 属性（版本号）的注意点：
1、必须是 \*.\*.\* 的格式
2、* 只能是数字

## npm 搜索包
1、去 npm 官网搜索关键字
2、日常工作和学习中积累用到的包，日后就可以快速知道自己需要在什么场合用什么包

## 下载安装包
```javascript
npm install || npm i
```

## require 导入 npm 包的基本流程
1. 在当前文件夹下 node_modules 中寻找同名的文件夹
2. 在上级目录中下的 node_modules 中寻找同名的文件夹，直至找到磁盘根目录
